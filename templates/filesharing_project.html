File Sharing Guide

A comprehensive step-by-step guide to securely sharing files over WireGuard VPN.

Prerequisites

Before starting, ensure the following:

  • WireGuard VPN is installed on both sender and receiver devices.
  • Both parties have established a secure VPN connection.
  • You can ping 172.20.0.1 to verify connectivity.

Download Python Server Script

Download the server script to enable file sharing over the VPN:

Download server.py

Sender Guide

Follow these steps to share your files:

  1. Enable WireGuard VPN and verify connectivity: 
    ping 172.20.0.1
  2. Select the folder you want to share and note its full path.
  3. Download and run the server script: 
    python3 server.py
  4. Enter the folder path when prompted by the script.
  5. Note the generated link (e.g., http://your-ip:8000/) and share it with the receiver.

Receiver Guide

Follow these steps to access shared files:

  1. Enable WireGuard VPN and verify connectivity: 
    ping 172.20.0.1
  2. Obtain the link from the sender (e.g., http://sender-ip:8000/).
  3. Open a web browser and navigate to the provided link.
  4. Browse the shared folder and download the desired files.
